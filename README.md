# SoundCue Marketplace

```bash
codex plugin marketplace add superche/soundcue --ref v0.1.0
codex plugin add soundcue@soundcue
```

```bash
codex plugin marketplace add /Users/bytedance/Developer/audio-patch-canvas-plugin/build/soundcue-marketplace
```
